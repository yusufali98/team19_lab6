CS7785_Lab6_Assignment Instructions

1. Use the "lab6_7785.yml" file to create a conda environment with the following command:
	conda env create --file lab6_7785.yml

2. Activate the conda environment using the following command:
	conda activate lab6_7785

3. Replace the content of the "test_images" folder with the images of the withheld dataset and the corresponding labels text file (ensure that the labels in the text file are in the numeric order)

4. Run the prediction.py file placed in the "src folder" using the following command:
	python prediction.py

5. Confusion Matrix and Accuracy will be printed in the terminal on running the script
